# AI-Assisted Multi-Disease Risk Intelligence System Using Explainable AI

## AI Capstone Project

This project predicts multiple disease risks using Machine Learning and Deep Learning techniques.

## Diseases Covered

* Diabetes Prediction
* Heart Disease Prediction
* Kidney Disease Prediction

## Technologies Used

* Python
* Scikit-learn
* XGBoost
* TensorFlow
* Keras
* SHAP
* Pandas
* Matplotlib

## Models Used

* Logistic Regression
* Random Forest
* XGBoost
* Artificial Neural Network (ANN)

## Explainable AI

SHAP is used to explain model predictions and feature importance.

## Project Features

* Multi-disease prediction
* ML + DL integration
* Explainable AI
* Model comparison
* Healthcare analytics
